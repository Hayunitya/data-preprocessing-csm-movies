# Data Preprocessing on CSM Movies Dataset (2014–2015)

Repository ini berisi proyek data preprocessing pada dataset **CSM (Conventional and Social Media Movies)** yang mencakup data film tahun 2014 dan 2015 dari UCI Machine Learning Repository. Proyek ini disusun sebagai tugas mata kuliah **Rekayasa Data**.

## Deskripsi Proyek
Proyek ini berfokus pada pembersihan dan penyiapan data melalui validasi nilai, penanganan missing values, pengendalian outlier, analisis korelasi antar atribut, serta feature engineering agar dataset siap digunakan untuk analisis lanjutan.

## Dataset
* **Sumber**: [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/424/csm+conventional+and+social+media+movies+dataset+2014+and+2015)
* **Dataset**: CSM Movies 2014–2015  
* **Jumlah data**: 217 instances, 12 fitur  

Dataset digunakan dalam kondisi mentah dan masih memerlukan preprocessing.

## Metode Preprocessing
* Validasi domain nilai
* Penanganan missing values
* Pengendalian outlier berbasis IQR
* Analisis redundansi atribut
* Feature engineering
* Evaluasi kebutuhan PCA

## Cara Menjalankan
1. Unduh file `Python_Project_Data_Preprocessing.ipynb` dari repositori ini.
2. Jalankan file menggunakan Google Colab atau Jupyter Notebook.
3. Jalankan setiap sel dari atas ke bawah untuk melihat hasil analisis dan grafik.
